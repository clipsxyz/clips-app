# Profile card illustrations (Passport page)

Drop SVG (or PNG) images here with these **exact filenames**. The app will show them on each card when the file exists; otherwise it shows the default icon.

| Filename          | Card title    | Suggested search for free illustrations      |
|-------------------|---------------|---------------------------------------------|
| `followers.svg`   | Followers     | "people group illustration free", "followers" – unDraw, Storyset |
| `following.svg`   | Following     | "connection network illustration", "social following"            |
| `bio.svg`         | Bio           | "profile user illustration", "about me" – unDraw "profile"       |
| `social-links.svg`| Social Links  | "social media links illustration", "share connect"               |
| `travel-info.svg` | Travel Info   | "suitcase travel illustration", "passport travel" (Airbnb-style)   |
| `location.svg`    | Location      | "map pin location illustration", "place location"                |
| `interests.svg`   | Interests     | "hobbies interests illustration", "heart like"                   |
| `country-flag.svg`| Country Flag  | "world globe illustration", "flag country"                       |

**Free illustration sites:** [unDraw](https://undraw.co), [Storyset](https://storyset.com), [Open Doodles](https://opendoodles.com). Prefer SVG for sharp scaling.
